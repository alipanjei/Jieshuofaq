Navigating with Jieshuo

How do I navigate on web pages with Jieshuo?

To navigate by controls and other elements with Jieshuo, navigate to:

  • “Program Settings”
  • “Action Settings”
  • and “Scrolling Settings”.

From there, check the checkbox which reads, “Browse by Control on Web Pages”. Swiping left or right enables you to cycle between the various navigational elements.

Then, to browse by link, for example, swipe left or right until Jieshuo reads “Links”. Then, swipe up to navigate to the prior link and down to navigate to the next link.

What is virtual navigation and detailed navigation? What's the difference?

While Jieshuo’s default navigation mode detects most items, there may be some cases where you’ll need to use detailed navigation. It allows you to access additional elements, such as buttons, which may not be accessible by default.

Virtual Navigation allows you to set markers, which you will be able to use later on to reach elements on the screen that can’t otherwise be reached by the focus.

What is a virtual screen?

Virtual Screen recognises the content of the screen but, unlike OCR, it allows you to activate any elements like links, buttons etc, by double-tapping on them.

What is automatic browsing, and how do I use it?

Automatic browsing causes Jieshuo to read all text and elements from your cursor’s current position. Typically, automatic browsing is useful for reading long articles or blog posts on the web where you’ll want to read all associated links.

What is reading mode, and how do I use it?

Reading mode reads just the text of a webpage, such as an article or blog post. However, it does not read any of the links, buttons, or ads which may be present in the text.

What is the difference between automatic browsing and reading mode?

Reading Mode allows users to read text in a straightforward way, skipping all elements such as links, images, etc. On the other hand, automatic browsing reads all the elements on the screen, including images, links, etc., reading mode skips all these other elements and only reads the text.

What is node browsing, and how do I use it?

Node Browsing makes it possible for the user to access parts of the screen that would otherwise be inaccessible. For example, it can be used when a group of controls is presented as a single element, requiring the user to click on a specific part of the element.

Editing with Jieshuo

How do I edit text?

To edit text within an edit field, swipe left then down to enter edit mode. Here, you can use navigational gestures to move through the text.

How do I cut, copy, and paste?

To cut, copy, and paste the whole block of text you’ve entered in an edit field, swipe up and right for the actions menu. Then, select the desired option from the menu.

For parts of text, do the following:

  • Swipe up then right for the actions menu
  • Select “Split Editing”
  • Select “More” from the strip of buttons at the bottom of the screen
  • Then, select the button just to the right of the “Search” button

Your text will appear as a list of checkboxes. Simply select the text you’d like to manipulate, and choose the appropriate option from the buttons at the bottom of the screen.

What is the Jieshuo clipboard, and how do I use it?

Any text that has been copied from a webpage or edit field  or translated from another language will appear in the Jieshuo clipboard.

Here, you can choose to add selected items to your favorites, paste previously copied text into an existing edit field, or manage your current clipboard items.

Customizing Jieshuo

How do I apply a custom sound theme?

Once you’ve downloaded and imported a sound theme, go to:

  • “Program Settings”
  • “Feedback Settings”
  • “Sound Themes”

Here, you can apply your sound theme of choice by checking the relevant box in the list of sound themes.

What are multi-part gestures, and how do I create them?

As the name suggests, multi-part gestures require more than one gesture in sequence to perform a task. An example might be the Down+Right gesture to open the main menu.

To create multi-part gestures, go to:

  • “Programme Settings”
  • “Action Settings”
  • “Multi-part Gestures”

Then, check the box that says “Use Multi-part Gestures”. Then you can assign gestures to your tasks of choice, selecting them from the list that appears on the screen.

What are edge gestures, and how do I create them?

Edge Gestures are performed at the left or right edge of the screen. When they’re set up, a small section of the screen at the edges is reserved for those gestures.

To create Edge Gestures, go to:

  • “Program Settings”
  • “Action Settings”
  • “Edge Gesture Settings”

Then, check the box that says “Use Gesture Settings”. You can then assign left or right gestures to tasks of your choice by selecting them from the list that appears on the screen.

Recently, a new option has been added that allows you to also use bottom edge gestures to go back, , go to home, open recent apps, open notifications bar, and more.

To set them up, check the relevant checkbox under the Edge Gesture Settings section, as indicated above.

What are custom gestures, and how do I create them?

Custom Gestures allow you to personalise CSR to better suit your preferences. To create custom gestures, go to:

  • “Program Settings”
  • “General Settings”

Then, check the box that says “Use Custom Gestures”.

After that you can assign your gestures of choice to perform tasks, by selecting them from the list in Custom Gesture Settings (the item right after Use Custom Gesture in General Settings).

What are Per-app gesture themes, and how do I create them?

Per-App Gesture Themes are sets of gestures that can be assigned to a specific app.

They can be created in two ways:

  • Open an app
  • Open the Main Menu and tap on More to get to the Multi-Column Menu
  • Tap on Edit Gestures, and assign the gestures of your choice to that specific app
  • Go to Program Settings, Action Settings, Per-App Gesture Theme and check the box relevant to the app.
  • Alternatively, Go to program settings, action settings, gesture settings, and then “Create Per App Gesture Theme” at the top right-hand corner of the screen
  • Next, choose an app from the list
  • assign the desired gestures
  • In Action Settings, tap on Per-App Gesture Theme, and check the box associated with the app you have customised.

Why ISN'T Jieshuo in the Play Store?

The process for putting Jieshuo in the Play Store has already been activated with the approval of the dev, but unfortunately it has been delayed due to the COVID-19 emergency. We apologise for the inconvenience.

Is Jieshuo secure?

Yes. Jieshuo has been tested for security by several competent people, including an independent IT security expert, and all found it to be perfectly   reliable and secure. No flaws were detected.

In the vast majority of cases, it should take no more than 24 hours for all Jieshuo licenses to be activated. However, due to time zones and other internet restrictions beyond our control, it can take slightly longer than this time frame. For this reason, please give us 48 hours for license activation before submitting an inquiry.

How can I get Jieshuo translated into my language?

Jieshuo has already been translated into a number of languages, and others are coming. All the work is done by volunteers, so if you want to have Jieshuo localised in your language and think you could translate it, you can join the dedicated Telegram group, obtain the strings and proceed. If you are not in a position to do the translation yourself, you should find someone else who can localise Jieshuo for you and other speakers of your language.

What is the Jieshuo camera, and how do I use it?

The Jieshuo Camera is a built-in camera that allows you to recognise a variety of objects, as well as faces, text, money and scenes.

Open the camera, press the Volume Down key till you reach the desired recognition option, they press the volume Up key to take a picture. After a few seconds, you’ll hear the result.

What's the difference between Jieshuo's voice assistant and Google Assistant?

Jieshuo’s Voice assistant can only perform a limited number of functions at the moment. It can open apps, call, tell time, current position, etc. However, it is possible to create a wide range of custom voice commands, also to perform multiple operations.

How do I contact the developer?

For all contact information, please go to “Program Settings” and then “Help and Feedback”. There, you’ll find Li’s email address.

Moreover, you’ll find links to jieshuo’s official Telegram and Whatsapp groups in both English and other languages. In these groups, you can contact Jieshuo’s community administrators, share Jieshuo tips and tricks, as well as find answers to questions not covered in the manual or FAQ.

Why does Jieshuo not work well with Samsung's default launcher, camera, and keyboard? What can I do about it?

As far as Samsung Launcher is concerned, there are issues with devices running Android versions up to 9. On Android 10, with the new version of One Ui Home, all problems have been solved. If you’re on an Android version prior to 10, you can install a third-party launcher such as Hyperion (which also provides Actions).

Samsung Camera, Keyboard and browser are currently not usable with Jieshuo, and can be replaced by other accessible camera apps (Open Camera, for example), and keyboards (for instance, Gboard).

How do I import plugins and other resources if automatic import doesn't work?

The easiest way to import a resource when the automatic method doesn’t work is to save the file to downloads, and then import it via a file manager (for example, Total Commander).

What's the difference between the international and regular versions of Jieshuo?

The international version of Jieshuo                 doesn’t contain the Chinese tts, and other services currently only available in China. Otherwise, both versions offer exactly the same features.
